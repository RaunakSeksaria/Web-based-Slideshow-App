# Slideshow based webapp    
This was developed in the Spring'24 semester at IIIT-H as part of the course project of the course "Intro-to-Software-Systems" by a team of 3 undergrads (Krish Agarwal, Manas Agrawal and Raunak Seksaria). 

The requirements are listed in [Course Project](Course%20Project.pdf)


## Description
Deployed a full-stack web-app, using HTML, CSS, JavaScript, Python(libraries used : Flask, PyJWT, MoviePy, Gunicorn), SQL

## How to install 
```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

Run using 
```bash
flask run
```
